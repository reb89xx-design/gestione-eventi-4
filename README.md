# Event Manager (Streamlit) - Prototipo

## Descrizione
Web app per la gestione eventi giornalieri per agenzie di spettacolo. Include:
- Calendario globale filtrabile
- Schede evento editabili
- Anagrafiche: Artisti, Format, Promoter, Risorse (DJ, Vocalist, Ballerine, Service, Tour Manager, Mascotte)
- Login leggero per team (3 utenti di esempio creati dal seed)

## Requisiti
- Python 3.9+
- Virtualenv consigliato

## Installazione
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
